# NguyenHoaiNam_Profile

Static profile website built with Bootstrap 5.

## Structure

- index.html
- projects.html
- assets/css/style.css
- assets/js/main.js
- assets/images/personal/
- assets/docs/

## Quick edit

1. Update your profile content in index.html.
2. Put your personal images inside assets/images/personal/.
3. Keep your avatar at assets/images/personal/Avt.jpg or update the image path in index.html.
4. Replace assets/docs/NguyenHoaiNam_CV.pdf with your real CV.

## New sections

- Download CV button on navbar and hero section
- Education and experience timeline
- Separate detailed project page: projects.html

## Run locally

Open index.html in a browser, or use a simple local server.
